ExcelPlus

A cross-browser solution to read an Excel file

Download this project as a .zip file Download this project as a tar.gz file

Cross-browser reader for Excel files

This solution can use ActiveX for old IE browsers, or XSLX.js (attention make sure you have a version that supports IE8).

It will relay on the FileAPI for modern browsers or on Flash for older browsers (using FileToDataURI project).

The idea is to combine everything to be easy to use.

Quick Start

You'll need to include the below files:

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js"></script>
<script type="text/javascript" src="dist/xlsx.js"></script> <!-- https://github.com/stephen-hardy/xlsx.js -- v2.0.0 with IE8 fix -->
<script type="text/javascript" src="dist/jszip.min.js"></script> <!-- https://github.com/Stuk/jszip -- v2.2.0 -->
<script type="text/javascript" src="dist/excelplus-2.0.js"></script>

Now we'll define a "load file" button in the HTML code:

<object id="file-excel"></object>

And finally the JavaScript code:

var oExcel = new ExcelPlus();
oExcel.openRemote({
  idButton:'file-excel',
  flashPath:'dist/swfobject/',
  readCallback:function(name,base64) {
    /* do something here -- see example and doc below */
  }
})

Make sure you ask for the last version of IE with the below metatag in the HEAD:

  <meta http-equiv="X-UA-Compatible" content="IE=Edge" />

Browser Compatibility

Tested with IE8, and all the modern browsers.

Live Example

Load an Excel file and see some info:

    Code for the previous example:

    var oExcel = new ExcelPlus();
oExcel.openRemote({
  idButton:'file-excel',
  flashPath:'dist/swfobject/',
  readCallback:function(name,base64) {
    var $list = $('#result');
    $list.append('<li>Number of sheets: '+this.nbSheets+'</li>');
    $list.append('<li>Name of the first sheet: '+this.getSheetNames()[0]+"</li>")
    // if there are several sheets you need to define which one to read:
    if (this.nbSheets > 1) this.selectSheet(1); // the first sheet is the one to read
    $list.append("<li>Value in the cell A1: "+this.read("A1")+"</li>");
    var range=this.read("A2:A3");
    $list.append("<li>Values in the cells A2:A3:<ul><li>A2: "+range[0][0]+"</li><li>A3: "+range[1][0]+"</li></ul></li>");
  }
})

    Quick Documentation

    new ExcelPlus(params) -> oExcel

    This is the master function for playing with a JS file.
    • {params}
      • useActiveX (Boolean | Optional)
        By default we'll try to use ActiveX if it's available

    oExcel.openRemote(params)

    This function permits to read a file in creating a upload button that will work with different browser versions.
    • {params}
      • flashPath (String | Default: "./")
        You must provide the path to the Flash files (swfobject.js, FileToDataURI.swf and expressInstall.swf)
      • idButton (String)
        The ID of the HTML button that we'll use to get the file
      • readCallback (Function | Optional | Default: function(name,base64){})
        The function that will be called when the file is read with two parameters (one: the file name, two: the base64 string of the file)
      • visible (Boolean | Optional | Default: false)
        If we use ActiveX then it's possible to make the Excel file we currently read visible on the screen

    oExcel.getSheetNames()

    Return an array of the sheet names (the key is the sheet #).

    For example:
        oExcel.getSheetNames(); // → [ 1 => "Sheet1", 2 => "Sheet2" ]

    oExcel.selectSheet(sheet)

    Select a sheet.

    For example:
        oExcel.selectSheet(1);
        oExcel.selectSheet("Sheet1");
    • sheet (String or Number)
      If it's a number, it must be greater than 0, and if it's a string it must be the name of a sheet.

    oExcel.readAll()

    Read the all usedRange. We assume that the content starts at A1.
    Returns a 2D array; the first one is the rows, and the second one is the columns.

    oExcel.read(range)

    → if it's a range then it returns a 2D array; the first one is the rows, and the second one is the columns
    → if it's a cell then it returns the value of the cell.

    For example:
        oExcel.read("A1"); // → returns a string
        oExcel.read("A1:B3"); // → returns an array
    • range (String)
      Unique cell (e.g. "C4") or a range of cells (e.g. "D2:E6")
